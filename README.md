# Blinkit Clone

A full-stack grocery delivery app built with FastAPI and Next.js.

## Stack
- **Backend:** FastAPI, SQLAlchemy, PostgreSQL, Cloudinary
- **Frontend:** Next.js, Tailwind CSS
- **Deployed:** Render (API) + Vercel (Frontend)

## Project Structure
- `/back-end` — FastAPI REST API
- `/front-end` — Next.js frontend
